# Changelog

## v1.0.1 - April 27, 2026

### Added

- **CONTRIBUTING.md** - contribution rules for consent, provenance, linguistic scope, and commercial use.
- **CODE_OF_CONDUCT.md** - project conduct rules centered on linguistic dignity and protection from anti-Blackness, misogynoir, dialect shaming, and extraction.
- **GOVERNANCE.md** - stewardship model that keeps the public standard open while protecting authority, certification, and private evaluation layers.

- **GitHub issue templates** - structured paths for benchmark cases, dialect variation, implementation questions, and harm or misuse reports.
- **Pull request template** - protective checklist for consent, sources, scope, and misuse risk.

## v1.0.0 — April 14, 2026

**The first AAVE linguistic respect layer for AI agents.**

### What's in this release

- **SKILL.md** — the full standard. Covers recognition, response, transcription, editing, and generation modes. Aspect marker table (habitual be, stressed BIN, completive done, be done, finna, stay, steady). Camouflaged forms. Syntactic features. Pragmatic features. Non-negotiable safety rules. Education and clinical justice. Eye dialect warning. AAE/AAVE/AASE continuum.

- **references/response-matrix.md** — operational rules by task type. What the agent should do when a user speaks in AAVE and asks a question, requests editing, asks for transcription, wants "professional" output, or asks for creative writing.

- **references/evaluation-and-red-team.md** — 6 failure modes (correction, mimicry, flattening, identity inference, professionalism bias, false-positive detection). 12+ red-team prompts. Minimum acceptance bar.

- **references/stt-and-normalization.md** — speech-to-text pipeline guidance. The fitna/fixing-to legal transcription case. Camouflaged form handling. Cadence preservation. Dual-artifact output pattern.

- **references/developer-resources.md** — CORAAL corpus, AAVENUE/ENDIVE/VALUE benchmarks, matched guise bias data, implementation checklist.

### Why this exists

No skill in the open agent ecosystem addressed linguistic respect for any dialect. This is the first. Built by Minista Jazz / Much Different World, reviewed across multiple model families, scholar-checked against Green 2002, Rickford 1999, Smitherman 1977, and Alim & Smitherman 2012.

### What's next

- Community contributions welcome — especially regional variation, additional camouflaged forms, and non-US Black English varieties
- Benchmark prompt suite expansion
- Integration guides for specific STT providers (Deepgram, Whisper, AssemblyAI)
