# Publication Boundary

Version: 1.0.2  
Status: Public skill package

This repository publishes the open AAVE Respect skill package for citation, review, implementation, and adaptation with attribution.

## Authored Contribution

The authored contribution is the AI-agent respect layer: an instruction package and supporting public guidance for recognizing and handling African American Vernacular English with linguistic respect.

This repository does not claim ownership of AAVE, African American English, Black English, Black language, or the scholarly and community lineages it cites.

## Public

This release includes:

- the `SKILL.md` instruction package
- public response matrix
- speech-to-text and normalization guidance
- evaluation and red-team guidance
- developer resources
- benchmark prompts
- stewardship, contribution, conduct, security, and trademark guardrails
- license and citation metadata

## Not Included

This release does not include:

- private language data
- private transcripts or speech samples
- internal model-tuning recipes
- private benchmark harnesses
- certification or compliance authority
- Much Different World application code
- registrant data
- commercial strategy or partner workflows

The open practice stands alone. Certification, endorsement, partnership, verified compliance, and private evaluation authority require separate written permission.
